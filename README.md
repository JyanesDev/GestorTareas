# GestorTareas

Sistema de gestión de tareas y proyectos desarrollado como proyecto
de aprendizaje profesional de Java.

## Tecnologías

- Java 21 (Eclipse Temurin)
- Maven 3.9+
- PostgreSQL 16 (Docker)
- Hibernate 6.4 / JPA 3.1
- Jakarta EE 10 (Servlets, JSP)
- JUnit 5
- SLF4J + Logback
- Apache Tomcat 10

## Requisitos

- JDK 21
- Maven 3.9+
- Docker Desktop
- IDE: NetBeans o IntelliJ Community

## Arrancar el entorno

```bash
# Levantar PostgreSQL + pgAdmin:
docker compose up -d

# Verificar:
docker compose ps
